## Description
Libft is a project on 42 School cursus where I have to code libc, the C standard library function. Below, are listed all functions i have implemented. <br /><br />

| Function | Description | 
| -------- | ----------- |
| | |
| ➔ ft_putchar_fd | write a character in a output |
| ➔ ft_putstr_fd | write a string in a output |
| ➔ ft_putnbr_fd | write a int number in a output |
| | |

<br />

## Installation
```bash
# Clone the repository
git clone https://github.com/be0x686172/libft/

# Enter in repository
cd libft

# Compile the executable file and run it directly
make -f makefiles/exe/Makefile

# Compile the static library
make -f makefiles/lib/Makefile
``
